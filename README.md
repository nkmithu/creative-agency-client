# Creative Agency
### MERN Stack Portfolio Project for Tech Agency 
[Visit the site](https://creative-agency-14d59.firebaseapp.com/)

## Key Features
* Tech Agency can promote their services and portfolio to clients.
* Publilc page consists with about, services, client feedback, contact.
* Admin see  orderd services from admin panel. 
* Admin can add services to public page abd take action based on order progress.
* Users can register with their information, can see their orderd services and their progress.
* Users can give feedback as well.
